# 开发指南

## 一、环境搭建

### 1.1 系统要求

| 要求 | 最低版本 | 推荐版本 |
|-----|---------|---------|
| Linux内核 | 5.15+ | 6.0+ |
| GCC | 9.0+ | 11.0+ |
| Clang | 13.0+ | 14.0+ |
| libbpf | 0.5+ | 1.0+ |

### 1.2 依赖安装

**Ubuntu/Debian:**
```bash
# 基础工具
sudo apt-get update
sudo apt-get install -y build-essential clang llvm gcc-multilib

# eBPF开发工具
sudo apt-get install -y libbpf-dev libelf-dev zlib1g-dev

# 其他依赖
sudo apt-get install -y pkg-config libjson-c-dev

# bpftool (如果系统包不可用)
git clone --recurse-submodules https://github.com/libbpf/bpftool.git
cd bpftool/src && make && sudo make install
```

**CentOS/RHEL:**
```bash
sudo yum install -y elfutils-libelf-devel zlib-devel
sudo yum install -y clang llvm
```

### 1.3 验证环境

```bash
# 检查内核版本
uname -r

# 检查BTF支持
ls /sys/kernel/btf/vmlinux

# 检查bpftool
bpftool version

# 检查libbpf
pkg-config --modversion libbpf
```

## 二、编译构建

### 2.1 完整编译

```bash
# 克隆项目
git clone https://github.com/ouyangyipeng/eBPF-Multi-Agent.git
cd eBPF-Multi-Agent

# 编译
make all
```

### 2.2 分步编译

```bash
# 仅编译BPF程序
make bpf

# 仅编译用户态程序
make user

# 清理
make clean
```

### 2.3 编译输出

```
build/
├── main.bpf.o      # BPF字节码
├── agent-monitor   # 可执行程序
└── *.o             # 中间目标文件
```

## 五、开发调试

### 5.1 BPF程序调试

```bash
# 查看BPF程序信息
bpftool prog show

# 查看BPF maps
bpftool map show

# 查看tracepoint
bpftool perf list

# 手动加载BPF程序
bpftool prog load build/main.bpf.o /sys/fs/bpf/test
```

### 5.2 日志调试

```bash
# 查看内核日志
dmesg | grep -i bpf

# 查看trace输出
cat /sys/kernel/debug/tracing/trace_pipe
```

### 5.3 常见问题

**问题1: BPF程序加载失败**
```
错误: Permission denied
解决: 使用root权限运行
```

**问题2: 找不到vmlinux.h**
```
错误: vmlinux.h not found
解决: make时会自动生成，或手动执行:
bpftool btf dump file /sys/kernel/btf/vmlinux format c > include/vmlinux.h
```

**问题3: Ring Buffer事件丢失**
```
错误: 事件丢失
解决: 增大Ring Buffer大小，或优化处理速度
```
